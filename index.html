FAQs

Creative Web Design is the art of combining design principles and technical skills to create visually appealing and functional websites that engage users.
This course is ideal for aspiring web designers, graphic designers, developers, entrepreneurs, or anyone interested in mastering the art of crafting compelling online experiences.
You will learn essential design principles, HTML, CSS, responsive design techniques, user experience (UX) design, and how to use popular design tools to create stunning websites.
No prior experience is required. This course is designed for beginners and will cover everything you need to know to get started in web design.
You will need a computer with internet access and design software such as Adobe Photoshop, Adobe XD, or Sketch. We will also use text editors for coding, such as Sublime Text or Visual Studio Code.
